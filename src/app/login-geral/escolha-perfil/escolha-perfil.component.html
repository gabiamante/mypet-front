Conheço e escolha seu perfil
Petclient é o cliente da nossa plataforma, ele irá usufruir dos serviços. Oferecemos contratação de serviços para andar com cachorros, busca produtos de petshops, divulgação de evento, hospadagens para os pets e ainda contato com clínicas veterinárias. Nesta modalidade é exclusivo cadastro de pessoa física.

PetHome são prestadores de serviço para hospedagem para os animalzinhos. Esse tipo de atividade pode ser feita por pessoas que façam esse serviço como um bico, portanto pessoa física, ou ainda faça em nome de alguma associação, sendo assim pessoa jurídica. Nesta modalidade são aceitos os dois tipos de perfis.

PetVet são exclusivos para clínicas veterinárias. Teremos clínicas veterinárias e rede de clínica veterinária. Nesta modalidade é exclusivo o cadastro de pessoa jurídica.

PetWalker são pessoas ou estabelecimentos que executem o serviço de andar com cachorro ou gatos. Nesta modalidade é para cadastro de pessoa jurídica ou física.

PetShop são exclusivos para os estabelecimentos que vendem produtos de pet. O cadastro pode ser de redes de lojas e lojas individuais. Nesta modalidade é exclusivo o cadastro de pessoa jurídica.
